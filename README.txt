# BD101_Spark_core_1
Task1: Find top 3 most popular hotels between couples. (treat hotel as composite key of continent, country and market). 
Implement using scala or python.

Task2: Find the most popular country where hotels are booked and searched from the same country. Implement using scala or python.
Create a separate application. Copy the application to the archive.

Task3: Find top 3 hotels where people with children are interested but not booked in the end. Implement using scala or python. 

You need installed java, javaC, JVM.

1)Scala program. For start enter commands:
    mnv clean
    mvn package

2)Get "BD101_Spark_core_1-1.0-jar-with-dependencies.jar" file at /target.
3)Java -jar BD101_Spark_core_1-1.0-jar-with-dependencies.jar <enter path to csv file>


P.S You can find test csv files at /src/main/resources
